# <a name="unknownattendeeconflictdata"></a>UnknownAttendeeConflictData

Элемент **UnknownAttendeeConflictData** представляет неразрешимого участника или участника, который не является пользователем, списком рассылки или контактом.

## <a name="remarks"></a>Заметки

Участник неизвестен, если он не может быть разрешен с объектом службы каталога Active Directory. Если не удается определить, что он является пользователем, группой или контактом, участник не будет разрешен. Например, посетитель не будет разрешен, если это публичная папка с включенной почтой.
  
Схема, описывающая этот элемент, расположена в виртуальном каталоге EWS на компьютере с MicrosoftExchange Server 2007 и установленной ролью сервера клиентского доступа.

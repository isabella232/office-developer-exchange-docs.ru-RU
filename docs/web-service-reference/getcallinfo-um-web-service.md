# <a name="getcallinfo-um-web-service"></a>GetCallInfo (веб-служба единой системы обмена сообщениями)

Элемент **GetCallInfo** определяет запрос на получения сведений о вызове. 
  
[GetCallInfo (веб-служба единой системы обмена сообщениями)](getcallinfo-um-web-service.md)
  
```xml
<GetCallInfo>
  <CallId/>
</GetCallInfo>
```

 **complexType**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[CallId (веб-служба единой системы обмена сообщениями)](callid-um-web-service.md) <br/> |Содержит идентификатор вызова, о котором нужно получить информацию.  <br/> |
   
### <a name="parent-elements"></a>Родительские элементы

Нет.

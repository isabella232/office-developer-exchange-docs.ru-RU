# <a name="openasadminorsystemservice"></a>OpenAsAdminOrSystemService

Элемент **OpenAsAdminOrSystemService** используется только для внутреннего использования. Этот элемент не используется клиентами. 
  
```XML
<OpenAsAdminOrSystemService LogonType="">
   <ConnectingSID/>
</OpenAsAdminOrSystemService>
```

 **OpenAsAdminOrSystemServiceType**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

|**Атрибут**|**Описание**|
|:-----|:-----|
|LogonType  <br/> |Не предназначен для использования клиентом.  <br/> |
   
### <a name="child-elements"></a>Дочерние элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|ConnectingSID  <br/> |Не предназначен для использования клиентом.  <br/> |
   
### <a name="parent-elements"></a>Родительские элементы

Нет.
  
## <a name="text-value"></a>Текстовое значение

Нет.
  
## <a name="remarks"></a>Заметки

Схема, описывающая этот элемент, находится в виртуальном каталоге IIS, в котором размещены веб-службы Exchange.Этот элемент появился в Exchange Server 2010 с пакетом обновления 1 (SP1).

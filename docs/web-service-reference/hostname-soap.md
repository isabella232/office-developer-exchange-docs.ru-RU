# <a name="hostname-soap"></a>Hostname (SOAP)

Элемент **Hostname** представляет часть имени хоста полного имени компьютера. 
  
```XML
<Hostname/>
```

 **строка**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[ProtocolConnection (SOAP)](protocolconnection-soap.md) <br/> |Представляет протокольное подключение веб-клиента сервера.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение элемента **Hostname** — это часть имени хостов полного имени компьютера.

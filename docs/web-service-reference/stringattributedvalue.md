# <a name="stringattributedvalue"></a>StringAttributedValue

Элемент **StringAttributedValue** указывает экземпляр в массиве атрибутов, связанных с элементом persona. 
  
```XML
<StringAttributedValue>
   <Value/>
   <Attributions/>
</StringAttributedValue>
```

 **StringAttributedValueType**

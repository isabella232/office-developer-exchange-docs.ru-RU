# <a name="hasblockedimages"></a>HasBlockedImages

Элемент **HasBlockedImages** указывает значение Boolean, которое указывает, заблокирован ли элемент изображения. 
  
```XML
<HasBlockedImages> true | false </HasBlockedImages>
```

 **Boolean**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Элемент](item.md) <br/> |Представляет универсальный элемент в Exchange магазине.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое **значение, истинное** для **элемента HasBlockedImages,** указывает на то, что элемент заблокирован изображения. Значение false **указывает** на то, что элемент не имеет заблокированных изображений. 
  
## <a name="remarks"></a>Заметки

Этот элемент появился в Exchange Server 2013.
  
Схема, описывающая этот элемент, находится в виртуальном каталоге IIS, в котором размещены веб-службы Exchange.

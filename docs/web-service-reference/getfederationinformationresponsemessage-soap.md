# <a name="getfederationinformationresponsemessage-soap"></a>GetFederationInformationResponseMessage (SOAP)

Элемент **GetFederationInformationResponseMessage** определяет ответ на запрос [операции GetFederationInformation (SOAP).](getfederationinformation-operation-soap.md) 
  
```XML
<GetFederationInformationResponseMessage>
   <Response/>
</GetFederationInformationResponseMessage>
```

 **GetFederationInformationResponseMessage**

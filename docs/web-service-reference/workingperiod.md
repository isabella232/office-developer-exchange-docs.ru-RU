# <a name="workingperiod"></a>WorkingPeriod

Элемент **WorkingPeriod** содержит рабочие дни и часы пользователя почтового ящика.

```xml
<WorkingPeriod>
   <DayOfWeek>...</DayOfWeek>
   <StartTimeInMinutes>...</StartTimeInMinutes>
   <EndTimeInMinutes>...</EndTimeInMinutes>
</WorkingPeriod>
```

 **WorkingPeriod**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[DayOfWeek (WorkingPeriod)](dayofweek-workingperiod.md) <br/> |Содержит список рабочих дней, запланированных для пользователя почтового ящика.  <br/> |
|[StartTimeInMinutes](starttimeinminutes.md) <br/> |Представляет начало рабочего дня для пользователя почтового ящика.  <br/> |
|[EndTimeInMinutes](endtimeinminutes.md) <br/> |Представляет собой конец рабочего дня для пользователя почтового ящика.  <br/> |
   
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[WorkingPeriodArray](workingperiodarray.md) <br/> |Содержит сведения о рабочем периоде для пользователя почтового ящика.  <br/> Ниже приводится выражение XPath к этому элементу:  <br/>  `/GetUserAvailabilityResponse/FreeBusyResponseArray/FreeBusyResponse/FreeBusyView/WorkingHours/WorkingPeriodArray` <br/> |
   
## <a name="remarks"></a>Заметки

Все элементы ребенка перечислены в последовательности, в которой они происходят. Схема, описывающая этот элемент, расположена в виртуальном каталоге EWS на компьютере с MicrosoftExchange Server 2007 и установленной ролью сервера клиентского доступа.

# <a name="targetautodiscoverepr-soap"></a>TargetAutodiscoverEpr (SOAP)

Элемент **TargetAutodiscoverEpr** представляет свойство **TargetAutodiscoverEpr.** Элемент **TargetAutodiscoverEpr** используется только для внутреннего использования. Этот элемент не используется клиентами. 
  
```XML
<TargetAutodiscoverEpr/>
```

 **anyURI**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[OrganizationRelationshipSettings (SOAP)](organizationrelationshipsettings-soap.md) <br/> |Представляет список отношений организации для одной организации.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовым значением для этого элемента является единый идентификатор ресурса (URI) для отношений организации.
  
## <a name="remarks"></a>Заметки

Этот элемент указывает URL-адрес сервера для внешней организации.

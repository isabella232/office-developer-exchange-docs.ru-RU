# <a name="dialstring-exchange-web-services"></a>DialString (Exchange веб-службы)

Элемент **DialString** представляет строку набора номера телефона, который вызван для воспроизведения элемента по телефону. Этот элемент обязательный. 
  
```xml
<DialString/>
```

 **строка**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[PlayOnPhone (Exchange веб-службы)](playonphone-exchange-web-services.md) <br/> |Представляет запрос на чтение элемента по телефону.  <br/> |
   
## <a name="remarks"></a>Заметки

Схема, описывающая этот элемент, находится в виртуальном каталоге IIS, в котором размещены веб-службы Exchange.

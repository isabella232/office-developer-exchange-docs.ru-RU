# <a name="domain-getfederationinformation-soap"></a>Domain (GetFederationInformation) (SOAP)

Элемент **Domain** определяет домен, который имеет доверие федерации. 
  
```XML
<Domain/>
```

 **строка**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Request (GetFederationInformation) (SOAP)](request-getfederationinformationsoap.md) <br/> |Представляет запрос [на операцию GetFederationInformation (SOAP).](getfederationinformation-operation-soap.md)  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение представляет доменное имя домена, который содержит доверие федерации.

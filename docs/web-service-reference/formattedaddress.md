# <a name="formattedaddress"></a>FormattedAddress

Элемент **FormattedAddress** указывает отформатированные значения отображения связанного почтового адреса. 
  
```XML
<FormattedAddress></FormattedAddress>
```

 **строка**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Value (PersonaPostalAddressType)](value-personapostaladdresstype.md) <br/> |Указывает сведения, связанные с почтовым адресом.  <br/> |
|[PostalAddress (PersonaPostalAddressType)](postaladdress-personapostaladdresstype.md) <br/> |Указывает почтовый адрес расположения.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение элемента **FormattedAddress** — это строковое значение, которое указывает форматированный адрес. 
  
## <a name="remarks"></a>Заметки

Этот элемент появился в Exchange Server 2013.
  
Схема, описывающая этот элемент, находится в виртуальном каталоге IIS, в котором размещены веб-службы Exchange.

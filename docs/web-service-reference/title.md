# <a name="title"></a>Заголовок

Элемент **Title** представляет название контакта. 
  
```xml
<Title/>
```

 **String**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[CompleteName](completename.md) <br/> |Представляет полное имя контакта.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение — это строка, которая представляет заголовок контакта.
  
## <a name="remarks"></a>Заметки

Схема, описывающая этот элемент, расположена в виртуальном каталоге EWS на компьютере с MicrosoftExchange Server 2007 и установленной ролью сервера клиентского доступа.

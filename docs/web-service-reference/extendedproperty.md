# <a name="extendedproperty"></a>ExtendedProperty

Элемент **ExtendedProperty** определяет расширенные свойства MAPI в папках и элементах. 
  
```xml
<ExtendedProperty>
   <ExtendedFieldURI/>
   <Values/>
</ExtendedProperty>
```

```xml
<ExtendedProperty>
   <ExtendedFieldURI/>
   <Value/>
</ExtendedProperty>
```

**ExtendedPropertyType**

## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[ExtendedFieldURI](extendedfielduri.md) <br/> |Определяет расширенное свойство MAPI для получения, набора или создания.  <br/> |
|[Values](values.md) <br/> |Содержит коллекцию значений для многоценного расширенного свойства MAPI.  <br/> |
|[Значение](value.md) <br/> |Содержит значение одно значение расширенного свойства MAPI.  <br/> |

## <a name="remarks"></a>Заметки

Схема, описывающая этот элемент, расположена в виртуальном каталоге EWS на компьютере, работающем под управлением Microsoft Exchange Server 2007, с установленной ролью сервера клиентского доступа.

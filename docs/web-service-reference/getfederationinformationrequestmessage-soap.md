# <a name="getfederationinformationrequestmessage-soap"></a>GetFederationInformationRequestMessage (SOAP)

Элемент **GetFederationInformationRequestMessage** готовит вызов на сервер для запроса данных конфигурации для службы маркеров безопасности (STS). 
  
```XML
<GetFederationInformationRequestMessage>
   <Request/>
</GetFederationInformationRequestMessage>

```

## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Request (GetFederationInformation) (SOAP)](request-getfederationinformationsoap.md) <br/> |Представляет запрос **GetFederationInformation.**  <br/> |
   
### <a name="parent-elements"></a>Родительские элементы

Нет.
  
## <a name="text-value"></a>Текстовое значение

Нет.

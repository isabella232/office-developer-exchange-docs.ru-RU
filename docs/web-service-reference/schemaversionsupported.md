# <a name="schemaversionsupported"></a>SchemaVersionSupported

Элемент **SchemaVersionSupported** содержит версию схемы манифеста, поддерживаемой клиентом. 
  
```XML
<SchemaVersionSupported />
```

 **строка**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

[GetAppManifests](getappmanifests.md)
  
## <a name="text-value"></a>Текстовое значение

Текстовое значение элемента **SchemaVersionSupported** содержит версию схемы манифеста, поддерживаемой клиентом. Это значение указывает, какие манифесты приложения должны быть возвращены клиенту в ответе. 
  
## <a name="remarks"></a>Заметки

Этот элемент появился в Exchange Server 2013 с пакетом обновления 1 (SP1).

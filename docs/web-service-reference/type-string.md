# <a name="type-string"></a>Type (строка)

Элемент **Type** указывает тип почтового адреса или номера телефона, например "Главная" или "Бизнес". 
  
```XML
<Type></Type>
```

 **строка**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

[Телефон](phone.md)  |  [PhoneNumber](phonenumber.md)  |  [Значение (PersonaPostalAddressType)](value-personapostaladdresstype.md)  |  [PostalAddress (PersonaPostalAddressType)](postaladdress-personapostaladdresstype.md)
  
## <a name="text-value"></a>Текстовое значение

Текстовое значение элемента **Type** — это тип почтового адреса или номера телефона. Например, для элемента **Type** ожидаются значения "Главная" или "Бизнес". 
  
## <a name="remarks"></a>Заметки

Этот элемент появился в Exchange Server 2013.
  
Схема, описывающая этот элемент, находится в виртуальном каталоге IIS, в котором размещены веб-службы Exchange.

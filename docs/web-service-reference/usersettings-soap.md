# <a name="usersettings-soap"></a>UserSettings (SOAP)

Элемент **UserSettings** представляет коллекцию параметров пользователя. 
  
```XML
<UserSettings>
    <UserSetting/>
</UserSettings>
```

 **UserSettings**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[UserSetting (SOAP)](usersetting-soap.md) <br/> |Представляет один параметр пользователя.  <br/> |
   
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[UserResponse (SOAP)](userresponse-soap.md) <br/> |Представляет ответ на запрос GetUserSettings для отдельного пользователя.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Нет.

# <a name="largeaudiencethreshold"></a>LargeAudienceThreshold

Элемент **LargeAudienceThreshold представляет** большой порог аудитории для клиента. 
  
```XML
<LargeAudienceThreshold/>
```

 **int**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[MailTipsConfiguration (MailTipsServiceConfiguration)](mailtipsconfiguration-mailtipsserviceconfiguration.md) <br/> |Содержит сведения о конфигурации службы для службы советов почты.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение представляет собой пороговое значение аудитории, которое указывает, что сообщение передается более чем одному человеку.
  
## <a name="remarks"></a>Заметки

Схема, описывающая этот элемент, находится в виртуальном каталоге IIS, в котором размещены веб-службы Exchange.

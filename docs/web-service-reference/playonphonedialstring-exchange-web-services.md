# <a name="playonphonedialstring-exchange-web-services"></a>PlayOnPhoneDialString (Exchange веб-службы)

Элемент **PlayOnPhoneDialString** определяет строку номера Play-on-Телефон. 
  
```XML
<PlayOnPhoneDialString/>
```

 **строка**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[UnifiedMessagingConfiguration](unifiedmessagingconfiguration.md) <br/> |Содержит сведения о конфигурации службы единой службы обмена сообщениями.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение для этого элемента — строка "Play-on-Телефон".
  
## <a name="remarks"></a>Заметки

Этот элемент обязательный.
  
Схема, описывающая этот элемент, находится в виртуальном каталоге IIS, в котором размещены веб-службы Exchange.

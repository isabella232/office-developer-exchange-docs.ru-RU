# <a name="server-soap"></a>Server (SOAP)

Элемент **Server** представляет альтернативный сервер почтовых ящиков. 
  
```XML
<Server/>
```

 **строка**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[AlternateMailbox (SOAP)](alternatemailbox-soap.md) <br/> |Представляет альтернативный почтовый ящик.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение элемента **Server** — это имя альтернативного сервера почтовых ящиков.

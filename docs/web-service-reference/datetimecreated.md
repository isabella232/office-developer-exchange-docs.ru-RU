# <a name="datetimecreated"></a>DateTimeCreated

Элемент **DateTimeCreated** представляет дату и время создания элемента в почтовом ящике. 
  
```xml
<DateTimeCreated/>
```

**DateTime**

## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

## <a name="text-value"></a>Текстовое значение

Текстовое значение представляет дату и время создания элемента в почтовом ящике.
  
## <a name="remarks"></a>Заметки

Использование объектов ответа календаря обновляет свойство [DateTimeCreated](datetimecreated.md) в связанном элементе календаря. Ожидается, что свойство **DateTimeCreated** останется неизменным. Например, пользователь A отправляет запрос на собрание пользователю B. Пользователь B принимает запрос собрания с идентификатором запроса собрания. Свойство **DateTimeCreated** связанного элемента календаря изменено. 
  
Схема, описывающая этот элемент, расположена в виртуальном каталоге EWS на компьютере с MicrosoftExchange Server 2007 и установленной ролью сервера клиентского доступа.

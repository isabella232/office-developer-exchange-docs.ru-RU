# <a name="displayname-nonemptystringtype"></a>DisplayName (NonEmptyStringType)

Элемент **DisplayName содержит** отображаемую именем нового контакта группы обмена мгновенными сообщениями или отображением имени новой группы обмена мгновенными сообщениями. 
  
```XML
<DisplayName></DisplayName>
```

 **NonEmptyStringType**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

[AddDistributionGroupToImList](adddistributiongrouptoimlist.md)  |  [AddImGroup](addimgroup.md)  |  [AddNewImContactToGroup](addnewimcontacttogroup.md)  |  [ImGroup](imgroup.md)
  
## <a name="text-value"></a>Текстовое значение

Текстовое значение элемента **DisplayName** — это отображение имени нового контакта группы мгновенных сообщений или отображаемого имени новой группы обмена мгновенными сообщениями. 
  
## <a name="remarks"></a>Заметки

Этот элемент появился в Exchange Server 2013.
  
Схема, описывающая этот элемент, находится в виртуальном каталоге IIS, в котором размещены веб-службы Exchange.

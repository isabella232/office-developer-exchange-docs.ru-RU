# <a name="ids"></a>Ids

Элемент **Ids** содержит массив идентификаторов определения часового пояса. 
  
```XML
<Ids>
   <Id/>
</Ids>
```

 **NonEmptyArrayOfTimeZoneIdType**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Id (TimeZone)](id-timezone.md) <br/> |Элемент, определяющие определение одного часовой пояса.  <br/> |
   
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[GetServerTimeZones](getservertimezones.md) <br/> |Определяет запрос на извлечение определений часовой зоны с Exchange сервера.  <br/> |

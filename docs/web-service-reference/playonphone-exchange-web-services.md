# <a name="playonphone-exchange-web-services"></a>PlayOnPhone (Exchange веб-службы)

Элемент **PlayOnPhone** представляет запрос на чтение элемента на телефоне. 
  
```xml
<PlayOnPhone>   <ItemId/>   <DialString/></PlayOnPhone>
```

 **PlayOnPhoneType**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[ItemId](itemid.md) <br/> |Представляет идентификатор элемента для воспроизведения на телефоне. Этот элемент обязательный.  <br/> |
|[DialString (Exchange веб-службы)](dialstring-exchange-web-services.md) <br/> |Представляет строку набора номера телефона, который вызван для воспроизведения элемента по телефону. Этот элемент обязательный.  <br/> |
   
### <a name="parent-elements"></a>Родительские элементы

Нет.
  
## <a name="remarks"></a>Примечания

Схема, описывающая этот элемент, расположена в виртуальном каталоге EWS на компьютере, работающем под управлением Microsoft Exchange Server 2010, с установленной ролью сервера клиентского доступа.

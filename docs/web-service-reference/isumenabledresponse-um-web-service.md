# <a name="isumenabledresponse-um-web-service"></a>IsUMEnabledResponse (веб-служба единой системы обмена сообщениями)

Элемент **IsUMEnabledResponse** определяет ответ на запрос на операцию [IsUMEnabled (веб-службы um).](isumenabled-operation-um-web-service.md) 
  
[IsUMEnabledResponse (веб-служба единой системы обмена сообщениями)](isumenabledresponse-um-web-service.md)
  
```xml
<IsUMEnabledResponse/>
```

 **Boolean**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

Нет.
  
## <a name="text-value"></a>Текстовое значение

Требуется текстовое значение Boolean. Ниже перечислены возможные значения.
  
- Верно
    
- Неверно

# <a name="uniquebody"></a>UniqueBody

Элемент **UniqueBody** представляет фрагмент HTML или простой текст, который представляет уникальное тело этого разговора. 
  
```XML
<UniqueBody BodyType=""/>
```

 **BodyType**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

|**Атрибут**|**Описание**|
|:-----|:-----|
|**BodyType** <br/> |Описывает, как хранится тело элемента в элементе.  <br/> |
   
#### <a name="bodytype-attribute"></a>Атрибут BodyType

|**Значение**|**Описание**|
|:-----|:-----|
|**HTML** <br/> |Преобразует все органы в HTML.  <br/> |
|**Text** <br/> |Преобразует все органы в обычный текст.  <br/> |

Нет.

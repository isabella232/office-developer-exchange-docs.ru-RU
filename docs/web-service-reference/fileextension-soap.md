# <a name="fileextension-soap"></a>FileExtension (SOAP)

Элемент **FileExtension** представляет расширение файла. 
  
```XML
<FileExtension/>
```

 **строка**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[SupportedFileExtensions (SOAP)](supportedfileextensions-soap.md) <br/> |Содержит список расширений файлов в расположении общего доступа к документам.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение элемента **FileExtension** является допустимым расширением файла для расположения общего доступа к документам.

# <a name="isnotsupported"></a>IsNotSupported

Элемент **IsNotSupported указывает,** нельзя ли изменить правило с помощью API управляемого кода. 
  
```XML
<IsNotSupported/>
```

 **Boolean**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Правило (RuleType)](rule-ruletype.md) <br/> |Представляет правило в почтовом ящике пользователя.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Значение true **указывает** на то, что правило нельзя изменить с помощью API управляемого кода. Значение false **указывает,** что правило можно изменить с помощью API управляемого кода. 
  
## <a name="remarks"></a>Заметки

Схема, описывающая этот элемент, находится в виртуальном каталоге IIS, в котором размещены веб-службы Exchange.

# <a name="isdeliveryreceiptrequested"></a>IsDeliveryReceiptRequested

Элемент **IsDeliveryReceiptRequested** указывает, запрашивает ли отправитель элемента квитанцию доставки. 
  
```xml
<IsDeliveryReceiptRequested/>
```

 **Boolean**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.

|**Элемент**|**Описание**|
|:-----|:-----|
|[RemoveItem](removeitem.md) <br/> |Удаляет элемент из хранилища Exchange.  <br/> |
|[Message](message-ex15websvcsotherref.md) <br/> |Представляет сообщение электронной почты Exchange.  <br/> |

## <a name="text-value"></a>Текстовое значение

Текстовое значение **true указывает,** что квитанция о доставке запрашивается у получателя элемента. 
  
## <a name="remarks"></a>Заметки

Схема, описывающая этот элемент, расположена в виртуальном каталоге EWS на компьютере, работающем под управлением Microsoft Exchange Server 2007, с установленной ролью сервера клиентского доступа.

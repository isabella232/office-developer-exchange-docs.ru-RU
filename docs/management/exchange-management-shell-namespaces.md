# <a name="exchange-management-shell-namespaces"></a>Пространства имен командной консоли Exchange

Сведения о пространствах имен для Exchange в Exchange.
  
**Применяется к:** Exchange Online | Exchange Server 2013 | Office 365
  
Exchange Приложения Для Exchange Online, Exchange Online в Office 365 или версии Exchange, начиная с Exchange 2013 г., взаимодействуют с Windows PowerShell средой с помощью типов, которые **Пространство имен System.Management.Automation.** В Exchange команды Management Shell используются типы из нескольких Exchange библиотек. В этом разделе приводится справочная информация о типах и интерфейсах из Exchange библиотек, используемых Exchange management Shell.
